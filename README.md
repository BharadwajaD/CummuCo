# CummuCo

Ride sharing webiste

## Setting up backend

``` bash
cd ./server
ACCSID="..." AUTH_TOKEN="..." node ./src/index.js
```

## Setting up frontend

```bash
cd ./cummuco-client
npm install
npm start
```
